# Fichier Audio Samba

La musique samba brésilienne est automatiquement jouée pendant l'easter egg (5 clics sur le logo).

## ⚠️ IMPORTANT : Télécharger le fichier audio

Le fichier `samba-music.mp3` doit être placé dans le dossier `public/` pour que la musique fonctionne.

## Méthode 1 : Script automatique (recommandé)

```bash
./scripts/download-samba-music.sh
```

## Méthode 2 : Téléchargement manuel

1. **Allez sur Pixabay** : https://pixabay.com/music/search/samba/
2. **Téléchargez** un fichier MP3 de samba brésilienne (libre de droits)
3. **Nommez-le** `samba-music.mp3`
4. **Placez-le** dans le dossier `public/`

## Méthode 3 : Utiliser une autre source

- **Bensound** : https://www.bensound.com/royalty-free-music/track/brazil-samba
- **Freesound** : https://freesound.org (recherchez "samba brazilian")
- **YouTube Audio Library** : Musique libre de droits

## Format requis :

- **Nom** : `samba-music.mp3` (exactement)
- **Format** : MP3
- **Durée** : 10 secondes minimum (sera coupé à 10 secondes)
- **Emplacement** : `public/samba-music.mp3`

## Pour utiliser votre propre fichier local :

1. Téléchargez ou créez un fichier audio de samba brésilienne
2. Nommez-le `samba-music.mp3`
3. Placez-le dans le dossier `public/`
4. Le système utilisera automatiquement votre fichier local en priorité

## Format recommandé :

- Format : MP3
- Durée : 10 secondes (ou plus, sera coupé à 10 secondes)
- Volume : Le volume est automatiquement réglé à 60%

## Sources recommandées pour de la musique libre de droits :

- **Pixabay** : https://pixabay.com/music/search/samba/ (recommandé - déjà configuré)
- Freesound.org (recherchez "samba" ou "brazilian music")
- YouTube Audio Library (musique libre de droits)
- Pond5 (musique libre de droits)
- Zapsplat (effets sonores et musique)
